# برگ ضمیمه فنی - سامانه جامع صندوق پژوهش و فناوری گیلان

این سند شامل لیست دقیق امکانات (Features) و تکنولوژی‌های به‌کار‌رفته در پیاده‌سازی پورتال می‌باشد.

---

## 📋 لیست امکانات (Features)

### ۱. سیستم احراز هویت و مدیریت کاربران (Authentication & User Management)
- **سطوح دسترسی چندگانه:** پشتیبانی از نقش‌های مدیر سیستم (Admin)، کارشناس سرمایه‌گذاری (Investment Employee)، کارشناس اداری (Administrative Employee) و مشتری/شرکت (Customer).
- **احراز هویت دو مرحله‌ای (2FA):** ورود امن با استفاده از رمز یکبار مصرف (OTP).
- **مدریت نشست‌ها (Session Management):** استفاده از JWT و کوکی‌های امن برای مدیریت نشست‌های کاربر.

### ۲. مدیریت جامع شرکت‌ها (Company Management)
- **پروفایل شرکت‌ها:** ثبت و مدیریت اطلاعات کامل ثبتی، تماس و آدرس.
- **مدیریت صاحبان امضا:** امکان ثبت و مدیریت صاحبان امضا (تا سقف ۲ نفر) برای قراردادها.
- **استعلام خودکار:** اتصال به وب‌سرویس **رسمیو (Rasmio)** برای دریافت و به‌روزرسانی خودکار اطلاعات شرکت‌ها.
- **پنل‌های اطلاعاتی:** بخش‌بندی اطلاعات به صورت تیم، محصول، بازار و مالی.

### ۳. سیستم گردش کار و خدمات (Workflow & Services)
- **واحدهای عملیاتی مجزا:** تفکیک فرآیندها بین واحد سرمایه‌گذاری و واحد اداری.
- **مدیریت درخواست‌ها:** امکان ثبت، پیگیری و مدیریت وضعیت درخواست‌های خدمات.
- **تخصیص هوشمند:** ارجاع خودکار درخواست‌ها به کارشناسان مربوطه.
- **رهگیری وضعیت:** نمایش مرحله‌به‌مرحله وضعیت درخواست به مشتری.

### ۴. سیستم فرم‌ساز پویا (Dynamic Form System)
- **طراحی فرم‌ها:** ایجاد فرم‌های سفارشی برای خدمات مختلف بدون نیاز به کدنویسی.
- **کنترل دسترسی:** تعیین سطح دسترسی به فرم‌ها (عمومی یا اختصاصی).
- **ذخیره‌سازی اطلاعات:** قابلیت ذخیره موقت (Draft) و بازیابی داده‌های وارد شده.

### ۵. تحلیل مالی هوشمند و هوش مصنوعی (AI Financial Analysis)
- **استخراج داده‌ها:** استخراج خودکار اقلام اطلاعاتی از فایل‌های اظهارنامه مالیاتی.
- **تحلیل هوشمند:** استفاده از **Claude AI** برای تحلیل صورت‌های مالی و ارائه پیشنهادات.
- **گزارش‌دهی:** تولید گزارش‌های تحلیلی خودکار بر اساس داده‌های مالی.

### ۶. سیستم تولید قرارداد خودکار (Automated Contract Generation)
- **مدیریت قالب‌ها:** امکان بارگذاری قالب‌های قرارداد (Word) در سیستم.
- **متغیرهای هوشمند:** پشتیبانی از بیش از ۱۰۰ متغیر آماده (اطلاعات شرکت، فرم‌ها، محاسبات سیستمی).
- **جایگزینی آنی:** تولید فایل نهایی قرارداد با جایگزینی خودکار مقادیر متغیرها در قالب انتخاب شده.
- **محاسبات پیچیده:** موتور فرمول‌نویسی برای انجام محاسبات مالی درون قراردادها.

### ۷. داشبوردهای تخصصی (Specialized Dashboards)
- **داشبورد مدیریت:** نمای کلی از عملکرد سیستم، کاربران و درخواست‌ها.
- **داشبورد کارشناسی:** کارتابل اختصاصی برای مشاهده و رسیدگی به درخواست‌های محوله.
- **داشبورد مشتری:** میز کار شرکت‌ها برای ثبت درخواست و مشاهده وضعیت پرونده‌ها.

### ۸. سیستم ارتباطات و پیام‌رسانی (Communication System)
- **چت داخلی:** امکان گفتگو بین کارشناسان و مشتریان روی هر درخواست.
- **ادغام با پیام‌رسان بله (Bale):** ربات هوشمند برای ارسال اعلان‌ها و پیگیری وضعیت از طریق پیام‌رسان بله.
- **اعلان‌ها (Notifications):** سیستم اطلاع‌رسانی درون‌برنامه‌ای و پیامکی.

### ۹. مدیریت اسناد (Document Management)
- **آرشیو دیجیتال:** بارگذاری، دسته‌بندی و مدیریت اسناد و مدارک شرکت‌ها.
- **OCR فارسی:** قابلیت تبدیل تصویر به متن برای اسناد فارسی با استفاده از Tesseract.js.

### ۱۰. امنیت و حفاظت (Security)
- **رمزنگاری:** هش کردن کلمات عبور با الگوریتم **Bcrypt**.
- **محدودیت نرخ (Rate Limiting):** جلوگیری از حملات Brute Force و DDoS.
- **حفاظت داده‌ها:** جلوگیری از حملات SQL Injection و XSS.

## 🛠️ تکنولوژی‌های پیاده‌سازی شده (Technology Stack)

### سمت کاربر (Frontend)
- **زبان:** TypeScript
- **کتابخانه اصلی:** React 18
- **استایل‌دهی:** Tailwind CSS + Shadcn/ui
- **مدیریت انیمیشن:** Framer Motion
- **مسیر‌یابی (Routing):** Wouter
- **مدیریت درخواست‌ها:** TanStack Query (React Query)
- **اعتبارسنجی فرم‌ها:** React Hook Form + Zod

### سمت سرور (Backend)
- **زبان:** TypeScript
- **محیط اجرا:** Node.js
- **فریم‌ورک وب:** Express.js
- **پایگاه داده:** SQLite (سازگار با PostgreSQL برای محیط عملیاتی)
- **ORM (مدیریت دیتابیس):** Drizzle ORM
- **امنیت:** Helmet.js, Passport.js, Express Rate Limit

### سرویس‌های خارجی و هوش مصنوعی (External APIs & AI)
- **هوش مصنوعی:** Anthropic Claude API (برای تحلیل‌های هوشمند)
- **اطلاعات ثبتی:** Rasmio API (برای استعلام شرکت‌ها)
- **پیام‌رسانی:** Bale Messenger API (ربات بله)
- **پنل پیامک:** SMS.ir API (برای ارسال OTP و اعلان‌ها)
- **پژوهش وب:** Perplexity API

### ابزارهای توسعه و زیرساخت (Dev & Infrastructure)
- **بیلد تول:** Vite (برای بیلد سریع و بهینه)
- **فرمت‌دهی کد:** Prettier & ESLint
- **مدیریت پکیج‌ها:** npm
- **معماری:** Monorepo (کلاینت و سرور در یک مخزن)
